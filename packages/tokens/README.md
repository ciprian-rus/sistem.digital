# @sistem-digital/tokens

Design tokens canonice pentru Sistem Digital, distribuite independent de framework.

## Instalare

```bash
pnpm add @sistem-digital/tokens
```

## TypeScript

```ts
import { color, spacing } from '@sistem-digital/tokens';
```

## CSS

```css
@import '@sistem-digital/tokens/css';
```

## JSON

```js
import tokens from '@sistem-digital/tokens/tokens.json' with { type: 'json' };
```

Pachetul se află în stadiu alpha. API-ul poate suferi schimbări incompatibile până la versiunea `1.0.0`.

Documentația și roadmap-ul sunt disponibile în repository-ul Sistem Digital.
