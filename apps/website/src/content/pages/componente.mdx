## Contractul canonic

HTML-ul semantic și clasele CSS reprezintă contractul de bază. JavaScript-ul este modular și opțional. React nu este obligatoriu pentru consumarea bibliotecii.

<CodeExample
  caption="Importul modulelor publice"
  language="css"

@import '@sistem-digital/components/navigation.css';
@import '@sistem-digital/components/content.css';
@import '@sistem-digital/components/interactive.css';`}
/>
