# Changesets

Orice pull request care modifică API-ul sau comportamentul unui pachet public trebuie să includă un changeset.

```bash
pnpm changeset
```

Alege pachetul afectat, tipul de versiune și descrie schimbarea pentru utilizatorii bibliotecii.

Nu este necesar un changeset pentru documentație, CI sau aplicații private precum `@sistem-digital/website`.
